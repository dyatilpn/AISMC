Домашнее задание №2: Разработка консольного приложения с применением ООП
Данный проект представляет собой консольное приложение на языке C# для управления данными предметной области «Больницы и Врачи». 
Приложение разработано в рамках учебного задания по дисциплине «Архитектура автоматизированных систем обработки информации и управления».
-------------------------------------------------------------------------------------------------------------------------------------------
===========================================================================================================================================


Вариант задания: №16
  Справочная таблица (сторона «один»): Больницы (Hospital).
  Основная таблица (сторона «много»): Врачи (Doctor).
  Числовой показатель: Стаж работы в годах (Experience). 
  Реализована валидация: стаж не может быть отрицательным числом.
  
Стек технологий
  Язык программирования: C# (.NET)
  Локальная база данных: SQLite 
  Библиотека для работы с БД: Microsoft.Data.Sqlite 
  
Основные функции
  Хранение данных: Вся информация сохраняется в локальном файле базы данных SQLite.
  Импорт тестовых данных: При первом запуске программы (если база данных пуста) происходит автоматическое чтение файлов hospitals.csv и doctors.csv для начального заполнения таблиц.
  Управление записями (CRUD):
    -Просмотр списка всех больниц и всех врачей.Добавление новых врачей (с предварительным выбором больницы из списка).
    -Редактирование данных существующих врачей.
    -Удаление врачей.
  Генерация отчетов: Отчеты формируются с использованием паттерна проектирования Fluent Interface (класс ReportBuilder).
    -Отчет 1: Полный список врачей с названиями больниц.
    -Отчет 2: Количество врачей в каждой больнице.
    -Отчет 3: Средний стаж врачей с группировкой по больницам.

Структура проекта:
  Program.cs — точка входа в приложение. Содержит реализацию интерактивного консольного меню в цикле и обработку пользовательского ввода.
  DatabaseManager.cs — класс, который инкапсулирует взаимодействие с SQLite. Выполняет создание таблиц, чтение CSV, CRUD-операции и выполнение SQL-запросов.
  ReportBuilder.cs — класс для построения и форматирования консольных отчетов. Использует цепочки вызовов методов (возвращающих this).
  Hospital.cs — класс-модель данных для таблицы больниц.
  Doctor.cs — класс-модель данных для таблицы врачей.hospitals.csv / doctors.csv — файлы с исходными данными (разделитель — точка с запятой ;).
